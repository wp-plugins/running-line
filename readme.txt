This plugin shows running line for a post of a chosen category.

== Description ==

If you would like to have your running line on the bottom of your blog, so this plugin is just for you! It gives you the possibility to show one of the posts from your chosen category by the running line. So download and enjoy!

In administration interface you can
1. order type of chosen category posts
2. select the direction of the running line
3. set running line scroll speed
4. set running line height
5. set running line font size
6. set running line text color
7. set running line border size
8. set running line border color
9. set running line background color
10. set running line padding

== Installation ==

1. Upload the running-line directory (including all files within) to the /wp-content/plugins/ directory
2. Activate the plugin through the Plugins menu in WordPress

== Frequently Asked Questions ==

= There is no running line after activation =

Make sure that the code <?php wp_footer(); ?> is included in your theme's footer.php file just before the </body> line.
